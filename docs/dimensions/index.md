# Dimensions

Each variable is defined over one of the following set of subcategories.

| Dimension name  | Description |
| ---- | ------------------------ |
| [`age`](/data-model/docs/variables/dimensions/age)|  Building construction year|
| [`building`](/data-model/docs/variables/dimensions/building)|  Building types|
| [`fuel`](/data-model/docs/variables/dimensions/fuel)|  Fuels|
| [`resource`](/data-model/docs/variables/dimensions/resource)|  Resources|
| [`tech_generation`](/data-model/docs/variables/dimensions/tech_generation) |  Electricity generation technologies|
| [`tech_heat`](/data-model/docs/variables/dimensions/tech_heat)| Heat production technologies |
| [`tech_hydrogen`](/data-model/docs/variables/dimensions/tech_hydrogen)  | Hydrogen production technologies |
| [`tech_liquids`](/data-model/docs/variables/dimensions/tech_liquids)| Liquid fuel production technologies |
| [`tech_methane`](/data-model/docs/variables/dimensions/tech_methane)| Methane production technologies |
| [`tech_storage`](/data-model/docs/variables/dimensions/tech_storage) | Storage technologies |
| [`trn_mode_freight`](/data-model/docs/variables/dimensions/trn_mode_freight) | Modes freight transport |
| [`trn_mode_private`](/data-model/docs/variables/dimensions/trn_mode_freight) | Modes passenger transport |
| [`use_elec`](/data-model/docs/variables/dimensions/use_elec) | Electricity end-uses |
| [`use_hydrogen`](/data-model/docs/variables/dimensions/use_hydrogen)| Hydrogen end-uses |
| [`use_liquids`](/data-model/docs/variables/dimensions/use_liquids)| Liquid fuels end-uses |
| [`use_methane`](/data-model/docs/variables/dimensions/use_methane)| Methane end-uses |
